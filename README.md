**SpaceX Falcon 9 Data Analysis — Guided Project (IBM Data Science Professional Certificate)**

This repository contains the notebooks I completed as part of a guided project in the **IBM Data Science Professional Certificate** on Coursera.
The exercise uses publicly available data about **SpaceX Falcon 9 launches** as a practical context to review and apply the main concepts introduced throughout the course.

The project is structured as a sequence of hands‑on labs, where the notebooks were partially pre‑written and I was required to complete the missing Python code.
The goal of the assignment is not to build a production‑ready system, but to practice the essential steps of a typical data science workflow — from data exploration to basic machine learning techniques — within a realistic aerospace‑related scenario.

This repository is included in my portfolio as part of my learning path in data science and as evidence of my completion of the IBM certification.
